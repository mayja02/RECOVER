Developed upon Esriâ€™s ArcGIS API for Javascript, the RECOVER web map client will enable end-users to consume a mashup of web services applicable to a specific wildfire event.

The graphical user interface will use a customized widget to enable the end-user to interact with the mapping application by assigning weighted values to selected layers such as fire intensity, fire severity, soils, soil moisture, and wildlife habitat. The custom values will then be automatically processed using a geoprocessing service. The results will enable the end-user to formulate a more informed rehabilitation and monitoring plan for the burned areas in question.
